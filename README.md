# osint
Simple osint tools :D
Command :
***Install python***

```apt install python -y```

***Installing git***

```apt install git -y```

***Clone repository***

```git clone https://github.com/blek1337/osint```

***Change dir***

```cd osint```

***Installing requirements***

```python -m pip install -r requirements.txt```

***Execute the script***

```python data.py```



Thanks for using :)
